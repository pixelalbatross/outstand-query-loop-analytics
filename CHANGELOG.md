# Changelog

All notable changes to this project will be documented in this file, per [the Keep a Changelog standard](http://keepachangelog.com/).

## [1.1.0] - 2026-07-07

### Added

- "Exclude current post" option on the Popular Posts variation, omitting the post being viewed from the query results on the front end and in the editor preview.

### Fixed

- Duplicate "GA4 property saved." notice when saving the settings page. The property picker is now part of the main settings form instead of a nested form, so saving shows a single "Settings saved." notice.

## [1.0.0] - 2026-07-06

- Initial release.
